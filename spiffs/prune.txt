# frixos OTA prune list
#
# Files that older firmware wrote to SPIFFS but current builds no longer ship.
# OTA reconcile only adds/updates manifest files - it never removes files that
# dropped out of the build - so these accumulate and can wedge a near-full
# filesystem. During an update the device downloads this list (its hash is
# covered by the signed manifest), then deletes any of these that still exist
# before downloading the new file set.
#
# One relative path per line (relative to /spiffs, forward slashes). Blank
# lines and lines starting with '#' are ignored. Only add files you are
# certain the current build does not use - a wrong entry is deleted on every
# device. Never list a file that is part of the current manifest.
