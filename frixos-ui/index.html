Firmware update

  1. Choose the firmware file (.bin) from your computer.
  2. Press “Upload file” to start the update.
  3. Wait for completion — don't power off or disconnect the device.
  4. The device restarts automatically when done.
Warning — uploading incorrect firmware may stop the device from working. Use only the firmware built for your model.

User file upload

  1. Choose the file (e.g. user-font.jpg) from your computer.
  2. Press “Upload file” to start.
  3. Wait for completion — don't power off or disconnect the device.
  4. Upload another file, or close this window.
