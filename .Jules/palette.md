## 2025-05-23 - Interactive Configuration Samples
**Learning:** Converting static visual samples (like font previews) into interactive buttons that programmatically update associated form fields reduces cognitive load and provides a more direct manipulation experience. It's especially effective when the underlying setting is a dropdown that can be hard to visualize.
**Action:** Always consider if visual previews can be made interactive. Ensure they have proper ARIA roles (button), keyboard accessibility (tabindex, Enter/Space listeners), and localized ARIA labels.
